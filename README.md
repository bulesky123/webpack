# webpack-router-singlePage
本应用主要采用webpack打包+前端 路由 实现一个单页demo（具体实现方法：router中用了一个插件director 监听hash值得变化，当hash值变化的时候会加载相应的HTML）
#安装
npm install (如若不行，请参照pakeage.json中依赖一个一个执行npm install 插架名)
#启动
npm start
# build
npm run build
